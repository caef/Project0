Project 0 Readme

This is an HTML prototype for a web application to manage a collection of records.

It includes an Index/Home page and three related pages: Albums, Artists and New Album. The buttons are not operational, with the exception of the navigation buttons in the navigation bar.

Files include
HTML (root directory)
  Index.html - Home page
  Albums.html - Album search and list display
  Artists.html - Artist Search and list display
  NewArtist.html - Add a new artist

CSS (root directory)
  styles.css - Partial Site CSS
  variables.scss - SASS styles definition (set if variables and related classes, selectors, etc)
   after compiling it generates
  variables.css -actual css
  variables.min.css- minimized variables.css
a few styles are defined in the HTML pages

IMAGES (\images directory)   Images displayed in the Home and Albums pages
The readme file is also stored in readme.md.txt

The HTML pages are built over Bootstrap, including the navbar facility
The bootstrap libraries are downloaded (CDN) and included the Javascript and Jquery libraries required by the navbar

All pages are responsive

All pages were built using Visual Studio 2017 Community over Windows 10 SCSS was compiled at save time using VS functionality

The navbar is implemented in every page. It could be implemented using a Layout page but may require a little Javascript to make the navbar actions dynamic
